# Flask App

Простое Flask-приложение, контейнеризированное с помощью Docker.

## Содержание репозитория

- `app.py` —  исходный код Flask приложения
- `Dockerfile` — файл для сборки Docker образа
- `.dockerignore` — файл для исключения ненужных файлов из образа
- `README.md` — инструкции по сборке и запуску

## Сборка Docker образа

Собрать локально образ можно командой:

```bash
docker build -t neshvilcabra/flask-app:1.0 .
